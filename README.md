# Pre-requisites
- Install [Node.js](https://nodejs.org/en/) version 20.3.0


# Getting started
- Clone the repository
```
git clone https://github.com/pnqcd/Weather-Forecast.git
```
- Install dependencies
```
cd react-express
npm install
```
- Build and run the project
```
npm start
```
  Navigate to `http://localhost:3000`
